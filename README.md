# CV Image Basics with OpenCV

This repository is my first practical Computer Vision lesson.  
The goal is to understand how images are represented inside a computer and how OpenCV handles image formats.

## Topics Covered

- What is an image in Computer Vision?
- Pixel matrix concept
- Image shape: height, width, channels
- BGR vs RGB difference
- RGB to Grayscale conversion
- Saving processed images
